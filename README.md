## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/AhmedHassan222/Sakan_Dashboard.git
2. **Install dependencies:**
   ```bash
   npm install
3. **Start the development server:**
   ```bash
   npm start

## Usage
